# Solvpath — Orders & Returns

## Setup

```bash
bun install
bun run dev
```

App runs at `http://localhost:3000` (or next free port).

Requires Node 20.19+ / 22.12+ and Bun.

## Scripts

- `bun run dev` — start dev server
- `bun run build` — production build
- `bun run lint` — lint
- `bun run format` — format with Prettier

## Limitations

- Due to time constraints, UI components were reused from an existing project rather than built from scratch.
